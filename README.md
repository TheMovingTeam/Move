# Move
Move is a universal transit app for android built to be fast, extensible, and extremely usable.

## How does it work?
The app gathers the stop, lines, and all data from all different providers from
a repository of your choice, alongside the capabilities of each such as Map
support or multiple estimations per line.

It keeps a local copy of this data up to date and then locally requests and
parses any data from each service when needed.

## Why was this made?
I was personally frustrated by the lackluster implementation from the
different public transit services in my city and surrounding territories.

I made it my own purpose to make the one public transit app to end them all because I
believe that public transportation is the best way to get around in a city and it breaks my heart that not as many people use it due to the poor user experience in things like ETAs or navigation.
